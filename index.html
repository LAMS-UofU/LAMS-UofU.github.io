ABOUT LAMS

The LIDAR Autonomous Mapping System (LAMS) is a prototype mapping robot device used to create a 3D Model for a given space. Designed and built from scratch a custom embedded PCB to store the point cloud data into an SD card. Designed and built a custom 3D modeling software to convert and produce the final 3D model image files. See details below.
