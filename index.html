ABOUT LAMS

The LIDAR Autonomous Mapping System (LAMS) is a prototype mapping robot device to create a 3D Model for a space. Designed and built from scratch a custom embedded PCB stores the point cloud data from the LIDAR scanner into an SD card (SDHC). Designed and built a custom 3D modeling software to convert the lam files from the SD car, and produce an obj file to use in the 3D viewer tool for a final 3D model image.
